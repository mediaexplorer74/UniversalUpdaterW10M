## Requirements
* W10M build 10240 or higher.
* CMD Injected using [CMD Injector v2.4.7.0](https://github.com/fadilfadz01/CMD.Injector) or higher.

## Available Updates
* 10.0.10549.4
* 10.0.10570.0
* 10.0.10586.107
* 10.0.14393.1066
* 10.0.15063.297
* 10.0.15254.603
* Offline package update
